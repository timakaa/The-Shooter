### Fast Start Github

```
git clone https://github.com/aphos-ai/the-shooter.git
cd the-shooter
npm install
```

then

```
npm run build
npm run start
```

or

```
npm run dev
```

---

### Fast Start Docker

```
docker pull timakaa/the-shooter
docker run -p 3000:3000 timakaa/the-shooter
```
